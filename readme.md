1.  In JavaScript, var, let, and const are used to declare variables, but they behave differently in terms of scope, re-declaration, and reassignment.

var:- var is the old way to declare variables in JavaScript.It has function scope.
A variable declared with var can be re-declared and reassigned.

let:-let was introduced in ES6 (2015).It has block scope (works inside {}).
It cannot be re-declared, but the value can be changed.

const:- const was also introduced in ES6.It has block scope.
It cannot be re-declared or reassigned after declaration.

2. The spread operator (...) in JavaScript is used to expand or spread elements of an array, object, or iterable into individual elements.

3. In JavaScript, map(), filter(), and forEach() are array methods used to loop through elements, but they work differently.

map():-
map() creates a new array.
It transforms each element of the array.

filter():-
filter() creates a new array with elements that pass a condition.

forEach():-
forEach() loops through an array but does NOT return a new array.
Mainly used to perform actions like printing or updating values.

4. An arrow function is a short and modern way to write functions in JavaScript, introduced in ES6 (2015).

5. Template literals are a way to write strings in JavaScript that allow embedding variables and expressions easily.
They use backticks (``) instead of quotes (" or ') and ${} for variables.
